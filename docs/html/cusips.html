cusips¶

Utilities for manipulating CUSIP security identifiers

Utilities for handling cusip check digits

@author: bart mosley @license: bg research llc, MIT License

References: https://www.cusip.com/static/html/webpage/pdf/CUSIPIntro_%207.26.2007.pdf

bgtools.cusips.cusipcheckdigit(cusip)¶

Calculate and return check digit. Assumes validated string–8 or 9 digits with alphas

bgtools.cusips.ischeckdigit(cusip)¶

True if string has nine digits with the ninth being the check digit

bgtools.cusips.validate_cusip(cusip)¶

Return validated full cusip, or None if validation fails validated cusip: uppercase alphas, with check digit validation: 8 digit core cusip, correct check digit if provided
